Melanoma Scan

 

Project Summary

Description

The app helps to identify melanoma from images taken by iPhones. The main melanoma detection part of the app was made using the ISIC archive data, and the machine learning algorithm I trained was inspired by the Inception v3 convolutional neural network.

The app also has many other features, from showing local environmental cancer risks to calculating likelygenetic risks based off of the user's phenotypes (shown in the demo below, and briefly in the sections below).

Project Background

I initially started to tackle this problem during the 2020 summer (the first of the 'covid summers'). I had plenty of time in hand as I was stuck indoors like most people so I had decided to take a few courses to learn about how neural networks and machine learning really works. That involved going over the fundamental mathematics and logic behind these algorithms (this deep learning specialisation was especially helpful).

Having learnt a lot of the fundamentals behind machine learning, I wanted to do something with it. So, I decided to try tackle a problem, and skin cancer detection was a problem that I was drawn to the most, as I could use what I had just learnt to make a positive societal impact.

Initial Model

Below is the very first model I made for this problem. This is an implementation of a neural network from complete scratch. I only really used numpy for vector maths, but the rest of the code did not use any libraries (apart from those to help me reformat images into vectors, display graphs etc). In this implementation I made functions for each node, the activations etc, and then connected them together so that there was a logistic output.

I do now know a lot of things I did wrong (such as wrong type of network, and a serious lack of data and computation power), but this project helped me learn those things.
